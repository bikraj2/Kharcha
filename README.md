## 	:toolbox: Getting Started


### :bangbang: Prerequisites

This project uses npm as package manager

```bash
 npm install --global 
```

### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/bikraj2/Kharcha.git)
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  flutter pub get 
```
